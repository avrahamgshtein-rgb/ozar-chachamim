# דמויות קשורות — רבינו בחיי בן אשר

| שם | סוג הקשר | תיאור הקשר |
|----|----------|-------------|
| רבי שלמה בן אדרת (הרשב"א) | רב מרכזי | מנהיג יהדות ספרד וראש האקדמיה בברצלונה; העביר לבחיי הן את המסורת ההלכתית והן את מסורות הסוד של אסכולת גירונה |
| הרמב"ן (נחמנידס) | השפעה מרכזית | הרמב"ן הנחיל לרשב"א את מסורות הקבלה של אסכולת גירונה; בחיי מתייחס אליו כמי ש"לימד אותנו את הדרך לעלות ובדרך האמת הציב אותנו" |
| הרמב"ם | השפעה פילוסופית | בחיי שואב ממנו בשכבת ה"רמז" (הפרשנות הפילוסופית), אך כופף את הרציונליזם תחת התיאולוגיה |
| רש"י | השפעה פרשנית | מרכזי בשכבת ה"פשט" של פירוש בחיי; מוכר כסמכות עיקרית בפרשנות הפשוטה |
| רבי יעקב בן אשר (בעל הטורים) | בן דור ומושפע | פרשן תורה ופוסק שחי בסמוך לתקופת בחיי; שאב מאותם מקורות; פירושו המקוצר מודפס עד היום בכל חומש סטנדרטי |
| דון יצחק אברבנאל | מושפע מאוחר יותר | מרבה לצטט את רבינו בחיי בפירושיו הנרחבים לתנ"ך, בעיקר בנוגע לאגדות מדרשיות |
| אברהם בן חייא | השפעה פילוסופית | מוזכר בשכבת ה"רמז" כסמכות מדעית-פילוסופית שתורתו "כבר מגולמת בכתבי הקודש" |
