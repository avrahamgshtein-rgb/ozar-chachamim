# Maharam Segal — מהר"ם סג"ל (רבי מאיר בן ברוך הלוי)

## Overview

רבי מאיר בן ברוך הלוי, המהר"ם סג"ל (c.1320–1390), היה מגדולי פוסקי אשכנז וממשיכי [[maharam-rothenburg|מהר"ם מרוטנבורג]]. נבדל ממהר"ם מרוטנבורג בשם ובזמן — שניהם "מהר"ם" אשכנזים, אך נפרדים לחלוטין.

- **Dates:** c. 1320–1390
- **Location(s):** Vienna / Germany
- **Historical context:** דור שאחרי פרעות 1348 (המגפה השחורה וגירושי הקהילות).
